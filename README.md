# AI 산학협력 관리 시스템

부산대학교 AI융합교육원의 산학협력·인턴십·취업연계와 SW중심대학 사업 성과를 관리하는 웹 시스템입니다. 기업 정보, 협력 현황, 실무자, 컨택 이력에 더해 연도별 정량실적과 사업 성과지표까지 한곳에서 기록·조회합니다.

이 문서는 어떤 기능이 있는지 사용자 관점에서 정리한 것입니다.

---

## 주요 기능 요약

- 기업을 목록/검색/필터/정렬로 찾고, 클릭해 상세 보기
- 기본정보, 인턴십·채용연계, 실무자, 컨택 이력을 한 화면에서 관리
- 기업명만 입력하면 외부 정보(소재지·홈페이지·업종·소개 등)를 자동으로 채워줌
- 연도별 산학협력·인턴십 정량실적을 추적하고, 달성률 추이를 한눈에 비교
- SW중심대학 사업 17개 성과지표를 연도별로 관리하고, 연차 엑셀만 올리면 자동 반영
- 기존 엑셀과 비슷한 입력 화면, 엑셀 파일 업로드/다운로드 지원
- @pusan.ac.kr 계정 로그인으로 접근 제한
- 관련 사이트(코딩역량센터·취업현황 등)로 바로 가는 패밀리 사이트 링크
- 여러 명이 동시에 수정해도 데이터가 덮어써지지 않음
- 모바일에서도 사용 가능

## 화면별 기능

### 1. 대시보드 (첫 화면)
교육원 전체 현황을 보여줍니다.

- 정량실적 현황판: 누적 산학협력·인턴십·참여학생·참여기업 요약과, 산학협력·인턴십 달성률의 연도별 추이 그래프(목표선 포함). 연도 탭을 누르면 해당 연도의 목표 대비 달성 게이지로 전환
- 관리 중인 총 기업 수, 인턴십 가능 기업, 채용연계 가능 기업, MOU 체결 완료 수. 카드를 클릭하면 그 조건이 적용된 기업 리스트로 이동
- 최근 컨택 이력. 항목을 클릭하면 팝업으로 전체 내용 표시

### 2. 기업 리스트
협력 기업을 표 형태로 보여줍니다.

검색 필터 (상단):
- 기업명 검색, 지역(부산/울산/경남/수도권/기타), 우선순위(A/B/C), 진행상태, AI 기술분야 키워드, MOU 체결 여부
- 비활성 기업 포함 여부 토글

협력 항목 다중 선택:
- 인턴십, 산학프로젝트, 교과과정혁신위원회, 특강연계, 채용연계, 표준현장실습기관, 해외교육, 가치확산, 창업, 기타
- 체크한 항목을 모두 만족하는 기업만 표시

정렬:
- 기관명(가나다/역순), 사업참여연도, 최근 미팅일, 우선순위, 진행상태, 최근 수정순

엑셀 다운로드:
- 현재 검색 조건의 결과를 .xlsx 파일로 저장

행을 클릭하면 해당 기업 상세 화면으로 이동합니다.

### 3. 신규 기업 등록
새 기업을 등록하는 화면입니다.

- 기업명·지역·유형·담당 교수·MOU·우선순위·진행상태 등을 입력
- 기업명 입력 시 중복 자동 확인. 이미 등록된 기업이면 경고
- 비활성된 동명 기관은 자동으로 다시 활성화되어 등록 (실수로 비활성 처리한 기업의 재등록 편의)
- ‘이름으로 자동 채움’ 버튼: 입력한 기업명으로 외부 정보 조회해 빈칸 자동 채움

### 4. 기업 상세
한 기업의 모든 정보를 4개 영역으로 보여줍니다.

**기업 기본정보**
지역, AI 기술분야, 주요산업, 담당 교수, 진행상태, MOU, 홈페이지, 소재지. 위키백과에서 가져온 소개글은 ‘보기’ 버튼으로 펼치는 형태.

**인턴십 및 채용연계 정보**
인턴십·산학·특강·채용연계·창업·기타 등 협력 항목. 요구 역량, 우대 전공, 수용 가능 인원도 기록 가능.

**실무자 목록**
이름·직책·연락처·이메일. 추가/수정/삭제 가능.

**컨택 이력 및 성과**
미팅·연락 기록을 날짜순으로 표시. 내용이 길면 자동으로 줄여서(…) 보여주고, 클릭하면 팝업에서 전체 내용 확인 + 수정·삭제.

상단 버튼:
- ‘기본정보 수정’: 기본 정보 편집
- ‘비활성’: 데이터를 보존한 채 목록에서 숨김. 필요시 ‘다시 활성화’로 복귀
- ‘완전 삭제’: 실무자·컨택 이력까지 영구 삭제 (두 단계 확인)

### 5. 엑셀 입력
기존 업체정보 엑셀과 비슷한 화면입니다. 직접 입력과 파일 업로드 둘 다 지원합니다.

**직접 입력 그리드**
한 줄에 [기업 정보 · 협력 사항 · 담당자 · 대표자]를 입력합니다. 셀을 더블클릭해 수정. 저장하면 자동으로 기업/협력/실무자 정보로 분리 저장됨. 같은 기업명으로 다시 저장하면 중복 없이 갱신.

**엑셀 파일 업로드**
기존 .xlsx 파일을 그대로 올리면 자동 적재됩니다.
- 헤더에 ‘기관명/기업명’ 컬럼이 있는 행을 자동 인식
- 같은 기관명은 갱신, 빈 칸만 채우고 기존 값은 보존
- 담당자/대표자 컬럼은 실무자로 자동 분리 저장

**등록된 모든 기업에 자동 채움**
한 번 클릭으로 전체 기업의 빈 칸을 위키·네이버·DART·공개 임금데이터로 채워줍니다. 기존 값은 절대 덮어쓰지 않습니다.

### 6. 산학·인턴십 실적
연도별 산학협력 프로젝트와 인턴십 실적을 관리합니다.

**산학협력 현황**
연도·학과(정컴/DS)·구분·유형·트랙으로 필터하고, 기업·교수·연구주제로 검색합니다. 현재 검색 조건의 결과를 엑셀로 다운로드할 수 있습니다.

**인턴십 현황**
연도별 인턴십 실적을 목록으로 보여줍니다.

**실적 업로드**
산학협력·인턴십 정량실적 엑셀(.xlsx)을 올리면 프로젝트·인턴십·학생이 자동으로 적재됩니다. 다시 올리면 기존 실적은 전체 교체되며, 기업·컨택 데이터에는 영향이 없습니다.

### 7. SW중심대학 성과
SW중심대학 사업의 정량실적을 연도별로 관리하는 화면입니다.

**성과 대시보드**
17개 성과지표를 영역별로 묶어 연도별 추이 그래프로 보여줍니다.
- 각 지표에 목표 대비 달성(목표선), 단위, KMAC 검증 결과(O=통과, X=보완요청)를 함께 표시
- 지표를 클릭하면 산출 근거(분자/분모/산출식) 확인
- 아래에 정밀 수치 성적표(목표/실적)와 총괄 원시값도 제공

**실적 엑셀 업로드**
연차별 SW중심대학 정량실적 엑셀을 올리면 미리보기 후 저장합니다. 해당 연도 데이터는 전체 교체되고, 새 연도를 올리면 표의 연도 축이 자동으로 늘어납니다.

> 참고: 이 자료(KMAC가 검증하는 공식 평가지표)는 메인 대시보드의 ‘정량실적 현황판’과 출처·정의가 다른 별개 자료입니다. 두 화면의 산학·인턴십 수치가 서로 다른 것은 정상입니다.

## ‘이름으로 자동 채움’ 출처

| 채워지는 정보 | 출처 |
|---|---|
| 기업 소개글 | 위키백과 |
| 소재지, 홈페이지 | 네이버 |
| 대표자, 업종, 설립일, 매출 규모 | 금융감독원 전자공시(DART). 상장사·외부감사 대상만 |
| 평균연봉, 신입사원 연봉 | 공개 임금 데이터. 공공기관·지방공기업 |

상장사는 DART의 공식 본사 주소를 우선 사용합니다.

한계: 평균연봉 같은 항목은 공공기관·지방공기업에서만 조회됩니다. 부산 지역 AI 스타트업처럼 공개 데이터가 거의 없는 기업은 소재지·홈페이지 정도만 채워지고 나머지는 직접 입력해야 합니다.

테스트해보기 좋은 기업명: 삼성전자(소개·업종·매출 등 풍부함), 부산도시공사·부산시설공단(평균연봉까지 채워짐).

## 안전장치

**로그인 접근 제한**
@pusan.ac.kr 계정으로 로그인한 사용자만 접근할 수 있습니다.

**동시 편집 보호**
두 사람이 같은 정보를 동시에 수정할 때, 나중에 저장하는 사람이 먼저 저장된 내용을 덮어쓰지 않습니다. ‘다른 사용자가 먼저 수정했습니다’ 안내가 뜹니다. (기존 구글 시트 방식의 데이터 소실 문제 해결)

**삭제 대신 비활성**
기업·실무자를 ‘비활성’으로 두면 목록에서만 숨겨지고 데이터는 남습니다. 연결된 실무자·컨택 이력 관계도 유지됩니다.

**완전 삭제는 두 단계 확인**
영구 삭제 전에 두 번 확인을 거칩니다.

**자동 번호 부여**
기업·실무자·컨택 이력에 C-001, P-0001, H-0001 같은 번호가 자동으로 매겨집니다.

**빠른 반응 속도**
같은 페이지를 다시 열 때 캐시에서 즉시 표시되고, 백그라운드에서 최신 정보를 갱신합니다.

## 현재 상태와 다음 계획

로그인(@pusan.ac.kr 계정 제한), 산학·인턴십 실적, SW중심대학 성과 관리까지 적용된 운영 단계입니다. 다음 항목들이 추가될 예정입니다.

2. 교내 서버 이전: 현재 외부 클라우드(Supabase)에서 교내 PostgreSQL로 이전 예정
3. 변경 이력 기록 및 자동 백업
4. 학생 이력 관리(학생 테이블) 추가
5. 자동 채움 범위 확대, 실제 업체 데이터 이관
6. (선택) 컨택 미응답 알림, 첨부파일
