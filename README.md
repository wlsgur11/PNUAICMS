# 부울경 AI기업 관리 시스템 (AI Biz Connect)

부산대학교 교육원에서 **부산·울산·경남 지역 AI기업의 인턴십·취업연계·산학협력**을 한곳에서 관리하기 위한 웹 시스템입니다. 기업 정보, 인턴십·채용연계 현황, 실무자, 컨택 이력을 체계적으로 기록하고 한눈에 확인할 수 있습니다.

이 문서는 **실제로 어떤 기능을 쓸 수 있는지**를 사용자 관점에서 설명합니다.

---

## 이 시스템으로 할 수 있는 일 (요약)

- 협력 기업을 **목록·검색·필터**로 빠르게 찾고, 클릭 한 번으로 상세 정보 확인
- 기업의 **기본정보 / 인턴십·채용연계 / 실무자 / 컨택 이력**을 한 화면에서 관리
- 신규 기업 등록 시 **기업명만 입력하면 외부 정보를 자동으로 채워줌** (소재지·홈페이지·업종·소개 등)
- 기존에 쓰시던 **엑셀과 똑같은 느낌의 입력 화면** 제공
- **여러 명이 동시에 수정해도 데이터가 덮어써지지 않도록** 안전장치 적용

---

## 화면별 기능 안내

### 1. 대시보드 (첫 화면)
교육원 전체 현황을 한눈에 보여줍니다.

- **관리 중인 총 기업 수 / 인턴십 가능 기업 / 채용연계 가능 기업 / MOU 체결 완료** 수가 카드로 표시됩니다.
- **최근 컨택 이력**이 아래에 정리되며, 각 항목을 클릭하면 **상세 내용을 팝업창**으로 자세히 볼 수 있습니다.

### 2. 기업 리스트
협력 기업을 표 형태로 보여줍니다.

- **기업명 검색**, **지역(부산/울산/경남/수도권/기타) 필터**, **협력 우선순위(A/B/C) 필터**로 원하는 기업만 추려서 볼 수 있습니다.
- 표에서 **기업명 · 담당 교수 · MOU 체결 여부 · 인턴십 가능 · 채용연계 가능 · 최근 미팅일 · 우선순위**를 바로 확인할 수 있습니다.
- 행을 클릭하면 해당 기업의 상세 화면으로 이동합니다.

### 3. 신규 기업 등록
새 기업을 등록하는 입력 화면입니다.

- 기업명·지역·유형·담당 교수·MOU·우선순위·진행상태 등을 입력합니다.
- **기업명을 입력하면 중복을 자동으로 확인**하여, 이미 등록된 기업이면 즉시 경고해 줍니다(중복 등록 방지).
- **‘이름으로 자동 채움’** 버튼을 누르면, 입력한 기업명으로 외부 공개 정보를 조회해 **소재지·홈페이지·주요 산업·매출 규모·기업 소개·평균연봉** 등 빈칸을 자동으로 채워 줍니다. (자세한 내용은 아래 ‘자동 채움’ 참고)

### 4. 기업 상세
한 기업의 모든 정보를 4개 영역으로 나누어 보여줍니다.

- **기업 기본정보** — 지역·AI 기술분야·주요산업·담당 교수·진행상태·MOU·홈페이지·소재지. 위키백과에서 가져온 **기업 소개글은 ‘보기’ 버튼을 눌렀을 때만** 펼쳐 보이도록 하여 화면을 깔끔하게 유지합니다.
- **인턴십 및 채용연계 정보** — 인턴십·산학·특강·채용연계 등 협력 항목을 표시하고, **요구 역량·우대 전공·수용 가능 인원**까지 기록할 수 있습니다.
- **실무자 목록** — 기업 담당자들의 이름·직책·연락처·이메일. **‘+ 추가’**로 실무자를 바로 등록합니다.
- **컨택 이력 및 성과** — 미팅·연락 기록을 날짜순으로 보여줍니다. **내용이 길면 자동으로 줄여서(…) 표시**되고, 항목을 클릭하면 **전체 내용을 팝업창**으로 자세히 볼 수 있어 화면이 늘어지지 않습니다.
- 상단의 **‘기본정보 수정’**으로 내용을 고치고, **‘비활성’**으로 목록에서 숨길 수 있습니다(데이터는 삭제되지 않고 보존됩니다).

### 5. 엑셀 입력
**기존에 쓰시던 업체정보 엑셀과 똑같은 느낌**으로 입력하실 수 있는 화면입니다.

- 한 줄에 **[기업 정보 · 협력 사항 · 담당자 · 대표자]**를 원본 엑셀 양식 그대로 넓게 펼쳐 입력합니다.
- 셀을 더블클릭해 바로 수정하고, ‘+ 행 추가’로 새 기업을 추가합니다.
- 저장하면 한 줄의 내용이 **자동으로 기업/협력/실무자 정보로 분리·정리되어 저장**됩니다. 같은 기업명으로 다시 저장하면 중복 생성 없이 기존 기업이 갱신됩니다.
- 익숙한 엑셀 방식이 편하신 분도 불편 없이 사용하실 수 있도록 만든 입력 방식입니다.

## ‘이름으로 자동 채움’ — 어떤 정보가 채워지나

기업명만으로 여러 공개 출처를 조합해 빈칸을 채워 줍니다.

| 채워지는 정보 | 출처 |
|---|---|
| 기업 소개글 | 위키백과 |
| 소재지 · 홈페이지 | 네이버 |
| 대표자 · 업종 · 설립일 · 매출 규모 | 금융감독원 전자공시(DART) — 상장사·외부감사 대상 |
| 평균연봉 · 신입사원 연봉 | 공개 임금 데이터 — 공공기관·지방공기업 |

**참고(한계):** 평균연봉 등은 공공기관·지방공기업에서만 확인되고, 부산 지역 AI 스타트업·일반 사기업은 공개 데이터가 거의 없어 자동으로는 소재지·홈페이지 정도만 채워지고 나머지는 직접 입력이 필요합니다.

> **자동 채움을 시험해 보실 때:** `삼성전자`(소개·업종·매출 등이 풍부하게 채워짐), `부산도시공사`·`부산시설공단`(공공·지방공기업이라 평균연봉까지 채워짐)으로 입력해 보시면 효과가 잘 보입니다.

## 안전장치 (기존 방식 대비 개선점)

- **동시 편집 보호** — 두 사람이 같은 정보를 수정 중일 때, 나중에 저장하는 사람이 먼저 저장된 내용을 덮어쓰지 않습니다. 대신 **‘다른 사용자가 먼저 수정했습니다’**라고 알려 줍니다. (기존 구글 시트 방식에서 발생하던 데이터 소실 문제 해결)
- **삭제 대신 비활성** — 기업을 지워도 실제로 삭제되지 않고 ‘비활성’으로 숨겨져, 연결된 실무자·컨택 이력이 사라지지 않습니다.
- **자동 번호 부여** — 기업·실무자·컨택 이력에 고유 번호(C-001, P-0001, H-0001 …)가 자동으로 매겨져 충돌이 없습니다.

## 현재 상태와 앞으로의 계획

현재는 **핵심 기능 시범판(MVP)**으로, 기능을 직접 보고 평가하실 수 있는 단계입니다. 아래 항목이 이후 추가될 예정입니다.

1. **로그인** — `@pusan.ac.kr` 계정만 접근하도록 제한 (현재는 링크로 누구나 접근 가능한 시범 상태)
2. **권한 등급** — 교수/조교/열람 등 역할별 접근·수정 권한 분리
3. **교내 서버 이전** — 현재는 외부 클라우드에 데이터가 있으나, 교육원 내부 서버로 옮기면 데이터가 교내에만 저장되어 유출 우려가 없습니다
4. **변경 이력 기록 및 자동 백업** — 누가 언제 무엇을 바꿨는지 기록, 사고 시 복원
5. **자동 채움 범위 확대**, **실제 업체 데이터 이관**(개인정보 포함 데이터는 교내 환경에서만)
6. (선택) 컨택 미응답 알림, 첨부파일 관리, 통계 대시보드 고도화
